## 功能列表

- 启动/关闭游戏
- 切换账号
- 收取荒原、魔精收取生产物品
- 每日心相（意志解析）
- 刷体力、最优材料关卡、活动关卡、吃糖选项
- 自动深眠域
- 自动醒梦域
- 银行购物
- 领取日常奖励
- 当期轶事
- 雨中悬想：迷思海（"寻思"周扫荡）
- 局外演绎：山麓的回音（肉鸽）
